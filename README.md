# Этот код добавит анимированное дерево бонсай в ваш терминал

## Как запустить:
1. Измените путь к файлам .txt в коде
2. Откройте терминал и перейдите к директиве bonsai
3. Вбейте в терминале:
```zsh
csc Program.cs
mono Program.exe
```
